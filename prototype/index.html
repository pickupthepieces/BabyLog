BabyLog 原型

静态 HTML 预览，无后端、无真实交互。按 UI设计书.md 实现，仅用于在手机上确认视觉、单手范围、夜间模式。

S-01 首页 孕周 / 今日摘要 / 快捷记录 / 趋势 › S-02 时间线 全部记录倒序，阶段/类型筛选 › S-03 B 超录入 拍照 + 结构化字段 + 单位校验 › S-04 成长曲线 胎儿 / 出生后曲线（静态 SVG） › S-05 设置 导出 / 配额 / 同步配置占位 ›

预览方式：
① 电脑浏览器开 prototype/index.html，缩到 ~390 px 看效果。
② 电脑跑 npx serve prototype，手机同 WiFi 访问 http://<电脑 IP>:3000。
③ 直接把 prototype 拷到手机本地（iCloud / 微信收藏）用 Safari 打开。
