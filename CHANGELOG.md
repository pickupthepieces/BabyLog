# Changelog

BabyLog 发版变更日志。每次 release tag 触发 CI 时，从这里提取对应版本的条目作为 GitHub Release 描述和 App 内更新提示。

格式：`## [versionName]` 一节对应一个 release，节内自由 Markdown。CI 用 `## [versionName]` 行作为切分锚。

---

## [Unreleased]

> 下次发版前在这里累积条目。打 tag 之前把 `[Unreleased]` 改成 `[X.Y.Z]`，再加新的空 `## [Unreleased]` 占位。

### Added
- 日视图新增"当前时刻"指示线（玫红横线 + 圆点，仅查看今天时显示，每分钟刷新）。
- 日视图支持左右横扫切换前一天/后一天，切换带整页滑动过渡；竖向滚动不受影响。

### Changed
- 今天摘要改为分类统计瓦片（品类色 + 大数值 + 明细），数值更新带动画。
- 日视图新增半小时虚线刻度，小时行高加大；同时刻多条记录改为水平胶囊排布，不再垂直堆砌。
- 记录列表卡片正文去掉与图标/角标重复的类型名和"待补充详情"占位，无详情时收为单行（首页列表、时间线、回收站、详情页一致）。
- 日视图与首页"最近记录"空状态补充操作引导文案。
- 孕期摘要"下次产检"瓦片提示改为指向产检表单的日期字段，不再要求在备注里写日期格式。
- 下拉刷新在同步未配置时改为轻提示，不再弹对话框打断操作。
- 筛选面板展开/收起、筛选 chips、视图切换按钮、列表行位移均加过渡动画。
- 时间线与资料库的搜索/日期筛选默认折叠为单行，点击展开；有筛选条件时折叠态显示"筛选中"提示。
- 时间线与首页记录行不再放行内编辑/删除按钮，统一点击进详情页后操作。
- 快捷栏图标全部重画为极简线条风格，小尺寸下更清晰。
- 全 app 提示条统一为同一组件与配色；同步设置状态改为卡片化键值行。
- 育儿期自动隐藏孕期专属入口（预产期计算器、增重曲线、LMP 推算等），档案字段随妊娠状态联动。
- 记录详情元数据时间改为人类可读格式；提醒动作措辞更清晰（停用/启用 vs 忽略）。

### Fixed
- 智能录入页改为标准浅色页头，修复返回按钮白底白字不可见的问题。
- 喂养记录摘要的母乳侧别从 L/R/BOTH 改为显示左侧/右侧/双侧。
- 子页面标题不再贴状态栏；详情页底部按钮不再被系统导航条遮挡。
- 移除资料库"出生证明·规划中"占位行与失效的"其他"筛选。

## [0.1.1] - 2026-06-11

### Changed
- 栗记主界面、录入表单、设置页与智能录入继续收敛为 Chestnut 风格。
- 记录摘要改为运行时根据结构化字段生成，减少 payload 冗余并兼容旧数据。
- Smoke 测试基建去重，DailyBabySummary 模型合并。

### Fixed
- 适配系统底部三键导航栏，避免底部保存按钮、免责声明确认按钮和智能录入按钮贴底或被遮挡。

## [0.1.0] - 2026-05-25

首个公开版本兜底基线条目。

### 包含主线
- 孕期记录：B超 / 产检（常规层 + 7 类 screening_*）/ 孕妈指标 / 胎动 / 宫缩 / FGR 曲线 / DueDate 计算器 / 提醒中心
- 育儿期骨架：feed / sleep / diaper / temperature / medication 等 11 个事件类型可记录可编辑
- AI 识别：B 超 OCR / 产检 OCR / 智能录入语音转写
- 家庭同步 S1-S5：HKDF + AES-GCM E2EE 推拉、附件文件加密上下行、24h 自动拉取、防回路写入
- 系统：首启免责门、本地备份导入导出、记录搜索、回收站、PocketBase 连接检测
- 工程：detekt 行数闸值、文件级拆分、smoke 覆盖核心链路
- App 自动更新：半自动 APK 升级（设置页"检查更新"→ 下载 → SHA-256 校验 → 系统安装器）
