# BabyLog 需求评估表

## Status

| 项目 | 内容 |
|---|---|
| 文档状态 | Draft |
| 日期 | 2026-05-15 |
| 评估范围 | 单机版 PWA MVP，兼顾后续腾讯云同步和 OCR |
| 优先级 | P0 必做；P1 强建议；P2 后续增强；P3 服务器或长期能力 |
| 基本诉求 | iOS / Android 通用是 P0；腾讯云服务器参数不写入仓库但接口必须预留 |
| 用户范围 | 第一目标是自己家庭使用；后续 MAY 小范围推荐给同事家庭 |
| 当前范围 | 当前 App MUST 仅支持单胎/单宝宝；多胎仅做数据模型预留，UI 不实现 |

## Scoring Rules

| 字段 | 规则 |
|---|---|
| 用户价值 | 1 低，5 极高 |
| 实现复杂度 | 1 简单，5 复杂 |
| 风险 | 1 低，5 高 |
| MVP | Yes 表示单机版第一轮应实现 |
| 接口预留 | Yes 表示即使不实现后端，也必须保留抽象边界 |

## State Handling

| State | Meaning | Required Behavior |
|---|---|---|
| draft | 表单未提交 | SHOULD preserve until submitted or discarded |
| saved | 已写入本地 | MUST appear in timeline |
| pendingSync | 本地已保存，等待同步 | MUST be tracked even when sync disabled |
| synced | 已和后端一致 | Future backend state |
| conflict | 本地和远端冲突 | Future backend state; default resolution is entity-level last-write-wins within same familyId |
| tombstoned | 软删除记录 | MUST keep deletedAt until future compaction |
| ocrPending | OCR 等待执行 | Future backend state |
| ocrNeedsReview | OCR 完成但需人工确认 | MUST NOT auto-save medical fields without review |
| storagePersisted | 浏览器已授予持久化存储 | MUST show in Settings when API is available |
| storageBestEffort | 浏览器未授予或不支持持久化存储 | MUST show export/install reminder |

## Error Handling

| Code | Condition | Required Response |
|---|---|---|
| STORAGE_UNAVAILABLE | IndexedDB 不可用 | 显示不可保存提示，禁止继续录入 |
| QUOTA_EXCEEDED | 本地存储空间不足 | 提示导出备份和删除附件 |
| INVALID_UNIT | 单位或数值范围异常 | 阻止保存并标出字段 |
| IMPORT_VERSION_UNSUPPORTED | 备份版本不兼容 | 阻止导入并保留原数据 |
| CAMERA_DENIED | 相机/相册权限失败 | 提供手动录入和重试 |
| OCR_UNAVAILABLE | OCR 接口不可用 | 回退到手动录入 |
| SYNC_UNAVAILABLE | 同步服务不可用 | 本地继续可用，变更保留 pendingSync |
| BACKEND_NOT_CONFIGURED | 后端地址未配置 | 不显示失败弹窗；设置页显示“未配置同步” |
| BACKEND_UNREACHABLE | 腾讯云后端不可达 | 本地继续可用；仅同步状态显示异常 |
| STORAGE_PERSIST_DENIED | 持久化存储未授予或不支持 | 继续可用；显示安装到主屏幕和定期导出建议 |
| ATTACHMENT_TOO_LARGE | 压缩后附件仍超过限制 | 阻止保存原图，允许仅保存结构化记录 |
| CROSS_BORDER_SYNC_UNCONFIRMED | 境外节点同步未确认 | 禁止启用 SyncAdapter |
